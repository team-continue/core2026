# パラメータチューニング

## MPPI コントローラ

設定ファイル: `core_mppi/param/default_params.yaml`

### コスト重み

MPPIの挙動を最も大きく左右するパラメータです。

| パラメータ | デフォルト | 効果 |
|-----------|-----------|------|
| `mppi.w_path` | `3.0` | 経路追従の忠実度。上げると経路に近づく |
| `mppi.w_goal` | `8.0` | ゴール到達の優先度。上げるとゴールに向かいやすい |
| `mppi.w_obstacle` | `18.0` | 障害物回避の強さ。上げると障害物から離れる |
| `mppi.w_control` | `0.3` | 制御入力のペナルティ。上げると動きが穏やかに |
| `mppi.w_smooth` | `0.8` | 軌道の滑らかさ。上げるとジグザグが減る |

!!! tip "チューニングの順序"
    1. まず `w_obstacle` で安全な回避を確保
    2. `w_path` と `w_goal` で経路追従とゴール到達のバランスを調整
    3. `w_control` と `w_smooth` で動きの滑らかさを仕上げる

### サンプリング設定

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `mppi.samples` | `160` | サンプル数。多いほど最適解に近づくが計算負荷増 |
| `mppi.horizon_steps` | `12` | 予測ステップ数。`dt` との積が予測時間（0.96秒） |
| `mppi.dt` | `0.08` | ステップ時間 [s] |
| `mppi.temperature` | `1.0` | 低い: 最良サンプル重視、高い: 平均的 |

### 速度制限

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `mppi.max_vx` | `1.0` | X方向最大速度 [m/s] |
| `mppi.max_vy` | `1.0` | Y方向最大速度 [m/s] |
| `mppi.max_wz` | `3.14` | 最大角速度 [rad/s] |

## PathFollower（PID制御）

設定ファイル: `core_path_follower/param/default_params.yaml`

### Cascade PID ゲイン

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `outer_kp` | `1.2` | 外側PID比例ゲイン（方位誤差→角速度参照） |
| `outer_ki` | `0.0` | 外側PID積分ゲイン |
| `outer_kd` | `0.15` | 外側PID微分ゲイン |
| `inner_kp` | `2.0` | 内側PID比例ゲイン（角速度追従） |
| `inner_ki` | `0.0` | 内側PID積分ゲイン |
| `inner_kd` | `0.05` | 内側PID微分ゲイン |

### 走行パラメータ

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `linear_speed` | `0.6` | 前進速度 [m/s] |
| `lookahead_dist` | `0.5` | ルックアヘッド距離 [m] |
| `goal_tolerance` | `0.15` | ゴール判定距離 [m] |

## コストマップ

設定ファイル: `core_costmap_builder/config/costmap_build_node.yaml`

### 膨張パラメータ

| パラメータ | デフォルト | 効果 |
|-----------|-----------|------|
| `inflation_radius_m` | `0.50` | 障害物周囲のLETHALゾーン幅。ロボット半径以上に設定 |
| `decay_margin_m` | `0.30` | LETHALゾーン外側の減衰幅。大きいと遠くの障害物も避ける |

### 点群フィルタリング

| パラメータ | デフォルト | 効果 |
|-----------|-----------|------|
| `min_z_m` | `0.25` | 地面を除去。地面が検出される場合は上げる |
| `max_z_m` | `1.60` | 天井等を除去 |
| `crop_xy_m` | `6.0` | 処理対象範囲。大きいと遠方も検出するが計算負荷増 |
| `voxel_leaf_m` | `0.05` | ダウンサンプリング。大きいと処理が軽くなるが精度低下 |
