    カスタムメッセージ（core_msgs）¶

    CAN.msg¶

    uint8 id
float32[] data

    CANフレームの抽象化。id はモータID、data は指令値。

    CANArray.msg¶

    CAN[] array

    複数のCANメッセージをまとめて送信。

    Path.msg¶

    std_msgs/Header header
PoseWithWeight[] pose

    重み付き経路。

    PoseWithWeight.msg¶

    geometry_msgs/Point position
geometry_msgs/Quaternion orientation
float64 distance_to_obstable

    障害物までの距離情報を持つ姿勢。
